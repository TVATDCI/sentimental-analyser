How to Use

To use any emoji, simply copy the markdown code and paste it where you'd like the emoji to appear in your text or project.

Interactive Copy (Optional)

For an interactive experience, consider adding a web-based emoji picker or a script like below to allow users to copy the emoji code:

<script>
function copyToClipboard(text) {
  navigator.clipboard.writeText(text).then(() => {
    alert(`Copied: ${text}`);
  });
}
</script>
